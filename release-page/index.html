使用步骤

DGHub 联动需要 Phigrab 和开启 DGHub 的设备处于同一局域网下。

安装 Phigrab

从 Release 下载适合设备的安装包。Android 可直接安装 APK，iOS 需处理签名。

启动 DGHub

并连接郊狼设备,确认 DGHub 设备和当前设备在同一 Wi-Fi 或可互通网络下，并检查防火墙设置。如未安装请下载Dghub

开启外部连接

在phigrab设置中开启外部连接。如果无效请检查dghub是否连接设备成功,并点击phigrab设置中的重新连接
